# AI Financial Assistant

AI Financial Assistant is a full-stack application that analyzes transaction data, generates spending insights, and flags unusual financial activity using anomaly detection and LLM-powered workflows.

## Features

- Upload and analyze transaction datasets
- Generate spending summaries and financial insights
- Detect suspicious or unusual transaction patterns
- REST API-based backend for analytics workflows
- Interactive frontend for financial data visualization

## Tech Stack

**Backend:** FastAPI, Python  
**Frontend:** React, Vite  
**Other Concepts:** REST APIs, anomaly detection, data analysis, LLM integration
